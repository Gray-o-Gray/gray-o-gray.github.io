# gray-o-gray.github.io

hexo使用记录

```
# 创建草稿
npx hexo new draft <title> 

# 将草稿发布为文章
npx hexo publish <title>

# 生成本地服务
npx hexo S

# 生成本地服务显示草稿
npx hexo S --draft

```

# Tools

1. hexo icarus
2. PicGo

# PicGo

图床工具，配合github.io使用非常舒适。

拖拽图片可自动上传github图床，并生成markdown形式的url，直接粘贴即可。
